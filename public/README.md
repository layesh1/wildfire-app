# public/

Static assets served directly by Next.js. Not application logic.

## Contents

| Path | What it is |
|------|-----------|
| `manifest.json` | PWA manifest (app name, icons, theme color, display mode) |
| `sw.js` | Service worker — handles web push notifications |
| `icons/` | PWA app icons in 12 sizes (72px–512px) |
| `gt-init.js` | Google Tag Manager initialization |
| `fire_events_map.json` | (in `data/`) Static fire event reference data |
| `*.png` / `*.jpg` | Hero images, caregiver photos, Flameo character images |
