# archive/

Original WiDS Datathon research application — the starting point from which Minutes Matter / WildfireAlert evolved.

## Contents

| Folder | What it is |
|--------|-----------|
| [`widsdatathon/`](./widsdatathon/) | The original datathon submission. Has its own detailed README. Contains the Streamlit multi-page app, the 62,696-row WatchDuty fire dataset with SVI and signal-gap columns, ML model artifacts (`evac_delay_model.pkl`, `fire_escalation_model.pkl`), analysis scripts, results, and visualizations. |

## Relationship to the current app

The research findings, equity framing (SVI disparity, 99.3% signal-gap), and evacuation timing statistics from the Streamlit app are the direct analytical foundation for the Next.js product built in this repository.

The full transformation story is documented in [`docs/ANISHA_transformation-story.md`](../docs/ANISHA_transformation-story.md).
