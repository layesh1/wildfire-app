# Analysis Scripts

Python scripts for exploratory data analysis (EDA).

**Run in this order:**
1. `eda_1_timeline_analysis.py` - Calculate evacuation delays
2. `eda_2_early_signals.py` - Identify predictive fire characteristics
3. `eda_3_geographic_patterns.py` - Map regional disparities

Results will be saved to `../04_results/` and `../05_visualizations/`
