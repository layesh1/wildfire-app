# hooks/

7 custom React hooks for shared client-side state and data fetching.

## Contents

| Hook | Purpose |
|------|---------|
| `useFlameoContext.ts` | Fetches and manages Flameo AI context data for the current user |
| `useFlameoNavContext.ts` | Flameo navigation context for routing-aware AI behavior |
| `useConsumerAlerts.ts` | Alert state management for the consumer hub |
| `useUserLocation.ts` | Geolocation — wraps the browser GPS API |
| `useResponderStationAnchor.ts` | Anchors responder UI to the user's active station |
| `useEnsureResponderStationFromProfile.ts` | Syncs station context from the user's profile |
| `useBreakpointMdUp.ts` | Responsive breakpoint detection (md and up) |
